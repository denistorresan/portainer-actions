# Portainer Actions

```bash
npm install
cp .env.example .emv
```

## Run

Basic Usage:

```bash
node update-stack.js
```

Advanced:

```bash
node update-stack.js --url https://localhost:9443 --apikey ptr_pC......5c/VU= --endpoint 2 --stack 3
```
